# AI Agent Instructions for this Project

## Project overview
- Simple static website with plain HTML.
- Pages: `index.html`, `intereses.html`, `contacto.html`.
- Local image assets stored in `Img/`.
- Content is written in Spanish and uses semantic HTML structure.

## What AI agents should know
- Do not introduce frontend frameworks or build tools.
- Keep changes simple and static: HTML, CSS, and images only.
- Prefer external styles in a new CSS file such as `styles.css` when adding styling.
- Maintain the current page structure and relative navigation links.
- Ensure accessibility and semantic correctness (e.g. `alt` text, headings, responsive viewport).

## Common tasks
- Improve page layout and styling with CSS.
- Fix HTML structure issues and ensure all pages share a consistent header/footer.
- Add or improve responsive design for mobile screens.
- Keep Spanish language content intact while editing structure.

## Notes for agents
- There is no build or test system in this workspace.
- Do not remove the `Img/` folder or local image references unless replacing them with valid alternatives.
- If creating a stylesheet, link it from all HTML pages using a relative path.
